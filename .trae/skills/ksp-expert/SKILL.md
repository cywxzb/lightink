# KSP Expert

This skill provides expert guidance on Kotlin Symbol Processing (KSP), including:

## Key Features

- **KSP Configuration**: Help set up KSP in Gradle projects, including plugin version matching and dependency configuration
- **Migration Support**: Assist in migrating from kapt to KSP, including dependency replacement and build script updates
- **Troubleshooting**: Diagnose and fix KSP compilation errors, processing order issues, and incremental build problems
- **Performance Optimization**: Provide recommendations for KSP performance tuning and build speed improvements
- **Integration Guidance**: Help integrate KSP with common libraries like Room, Glide, and Dagger

## When to Invoke

- When encountering KSP compilation errors
- When migrating from kapt to KSP
- When experiencing build performance issues with KSP
- When configuring KSP with specific libraries
- When needing to understand KSP processing order or incremental build behavior

## Common Issues Addressed

- "Element references a type that is not present" errors
- KSP plugin version mismatches
- Processing order issues with multiple KSP processors
- Incremental build failures
- Memory usage optimization for KSP
- Integration with Android Gradle Plugin

## Example Usage

**Scenario 1: KSP Compilation Error**
- Identify the root cause of KSP processing errors
- Provide specific fixes for type reference issues
- Suggest configuration changes to resolve dependency conflicts

**Scenario 2: kapt to KSP Migration**
- Identify all kapt dependencies that need to be replaced
- Update build scripts with correct KSP dependencies
- Verify the migration works correctly

**Scenario 3: Performance Optimization**
- Analyze KSP build times
- Suggest incremental build improvements
- Provide memory optimization recommendations

This skill is your go-to resource for all KSP-related issues and optimizations.
